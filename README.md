# Magicmark Tap

## How do I install these formulae?

`brew install magicmark/tap/<formula>`

Or `brew tap magicmark/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "magicmark/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
